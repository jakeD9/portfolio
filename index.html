MonoBot

Back-end Application

A Discord bot to help moderate server messages, and keep tabs on your staff. Also linked to Google Sheets.

This application was made for a friend who moderates a Discord server with over 3000 users. I was asked to create something that could generate a log of moderator activies to a spreadsheet. This was my first application built with a massive API, and Google Scripts. It was also my first time really diving into project architecture with Node's ability to export specific batches of code as modules and importing them elsewhere. I'm really satisfied with how this turned out.

Explore More

Front-End Application

A basic travel app to help inspire your next vacation or mini road trip.

Originally my first ever web project, Explore More has since been refactored using Sass, vanilla CSS, and new ES6 methods for DOM manipulation. It was a lot of fun going back and rewriting the first ever project using new things I've learned. One problem we always ran into was CORS issues when trying to talk to certain APIs. After much research and educating myself on web security, I've made my own "proxy" server of sorts that handles all third party API requests over HTTPS, just to make things easier in the future!
